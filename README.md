# Movie Finder Bot

Telegram бот для поиска фильмов и сериалов через TMDB API.

## Возможности
- 🔍 Поиск по жанрам, странам, году, рейтингу
- 🎲 Случайный фильм
- 🏆 Топ фильмов/сериалов
- ⭐ Избранное
- 📜 История поиска
- 🎲 Похожие фильмы
- 📝 Логирование диалогов

## Команды
- `/start` - главное меню
- `/help` - справка
- `/info <название>` - информация о фильме
- `/similar <название>` - похожие фильмы

## Технологии
- Python
- pyTelegramBotAPI
- TMDB API
- Flask

## Деплой
Бот размещён на Render.com и работает 24/7.
